# How To Rank on Google With OpenClaw

## Summary

Matthew Berman's complete SEO system built on OpenClaw — an automated SEO agent that runs weekly for $50/month, handling discovery, content creation, link building, and technical monitoring.

## Six Skills (Modules)

### 1. SEO Agent (Discovery + Monitoring)
- **seo-discover:** Find keywords in "strike zone" (positions 5-20)
- **seo-monitor:** Weekly ranking snapshots, track movement
- **seo-compete:** Competitor gap analysis

### 2. SEO Forge (Content Engine)
- Brand interview (8 questions) builds voice profile
- "Only I Can Write This" test — flags generic content
- Anti-AI-Overview by design (goes deeper/personal)
- Psychology frameworks: Puppet Strings, Scroll Traps, Care To Click
- Hub-and-spoke topical authority

### 3. SEO Images
- 6 generation styles, 21 presets
- Blog heroes, infographics, comparison charts
- Matched to content brief and voice profile

### 4. SEO Links (Backlink Engine)
- **link-mine:** Competitor backlink opportunities
- **link-mentions:** Unlinked brand mentions
- **link-broken:** Dead link opportunities
- **link-internal:** Site audit for orphan pages
- **link-prospect:** Resource page discovery

### 5. SEO Health (Technical)
- **health-speed:** PageSpeed Insights, Core Web Vitals
- **health-crawl:** Broken links, redirects, meta tags
- **health-images:** Alt text, file sizes, formats

### 6. SEO Checklist
- Meta tags, schema markup, llms.txt

## The Workflow (Real Week)

**Monday:** Discovery, monitoring, health checks run automatically → Telegram report at 7am

**Tuesday:** Agent writes supporting article → Draft + images by lunch → 15-min review → Publish

**Wednesday:** Internal link audit + unlinked mention outreach

**Thursday:** Competitor backlink mining + outreach

**Friday:** Competitor gap analysis → feeds next Monday's discovery

## Key Insights

- **Strike zone:** Positions 5-20 (4x click increase possible)
- **Interview is everything:** 15 minutes = highest leverage activity
- **Content without links is a Hail Mary**
- **Weekly monitoring catches climbers early** (2-3 week window to act)
- **Hub-and-spoke accelerates rankings** across entire cluster
