## 🔑 Key Capabilities (Nov 2024 Update)

### 1. AI Brief

**What It Does:**
Automatically generates tailored briefing documents for any customer situation using natural language prompts.

**Use Cases:**
- **Account Handoffs:** Automatically generate comprehensive account briefs when passing customers between teams
- **Win-Loss Reports:** Summarize deal outcomes with key learnings
- **Meeting Prep:** Create pre-meeting intelligence briefs
- **Custom Templates:** Define exactly what information you need summarized

**How It Works:**
1. Select or create template using natural language
2. Specify information needed and format
3. Gong analyzes all historical account/deal data
4. Generates high-quality brief in under 60 seconds

**Before vs After:**
- **Before:** Sellers spent considerable time creating inconsistent account briefs
- **After:** Every seller generates high-quality briefs in under a minute

**Result:** Better account handoffs, fewer missed opportunities, happier customers.

### 2. AI Scorecard Answers

**What It Does:**
Brings automation to sales coaching by automatically answering scorecard questions from call recordings.

**How It Works:**
1. Enablement leader creates scorecard
2. Creates natural language prompt using "Ask Anything" (e.g., "Did the rep confirm next steps?")
3. Gong AI runs prompt on call recording
4. Automatically finds answers and pre-populates key points
5. Manager reviews AI-suggested feedback

**Benefits:**
- Saves time on full call reviews
- Provides efficient, accurate feedback
- Raises rep performance faster
- Standardizes coaching across team

**Example Prompts:**
- "Did the rep confirm the next steps?"
- "Was the decision-making process discussed?"
- "Did the rep handle objections effectively?"
- "Was a timeline established?"

### 3. Gong Anywhere

**What It Does:**
Brings Gong AI directly into the tools where reps already work.

**Current Integrations:**
- Gmail inbox
- Salesforce
- Microsoft Outlook (coming soon)

**Benefits:**
- Eliminates app-switching
- Provides instant insights and context
- Enables personalization for high-quality outreach
- Combats productivity drain from siloed systems

### 4. Revenue Analytics

**New Capabilities (Nov 2024):**
- Define metrics and targets
- Create dashboards in Gong
- Single source of truth for pipeline management

**Pipeline Analytics:**
- Brings insights directly into pipeline-building workflows
- Previously required CRM, BI tools, or spreadsheets
- Now actionable within Gong interface

**Benefits:**
- More proactive and strategic growth
- Better opportunity identification
- Accurate forecasting
- Reliable revenue delivery
